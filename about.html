About This Notebook

This site serves as a technical notebook for explorations in Mathematics, Computer Science, and Competitive Programming. The content aims for rigor, clarity, and precision.

Topics Covered

Mathematics

Number theory, analysis, combinatorics, graph theory, and abstract algebra. Articles include formal proofs, theorem statements, and mathematical constructions.

Computer Science

Algorithms, data structures, complexity theory, systems design, and formal methods. Emphasis on correctness, invariants, and rigorous analysis.

Competitive Programming

Solutions and techniques for algorithmic contests. Includes implementation details, complexity analysis, and problem-solving strategies.

Technical Details

This site uses KaTeX for mathematical typesetting, supporting both inline and display mathematics. Code examples are syntax-highlighted for readability. The design prioritizes clarity and accessibility.
